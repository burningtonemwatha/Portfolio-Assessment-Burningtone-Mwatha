# Burningtone Mwatha - Portfolio Website

This is my personal portfolio website built with **HTML, CSS, JavaScript, Bootstrap, and PHP (for some projects)**.  
It showcases my profile, background, and featured projects related to **cloud computing, web development, and cybersecurity**.

---

## Overview

The website has two main pages:
- **Home (index.html):** Introduces who I am, my goals, and highlights my major projects.
- **About (about.html):** Contains my education, certifications, interests, and a simple contact form.

It’s fully responsive, has a light/dark theme toggle, and a clean modern interface.

---

## Features

- Responsive layout (works on phones, tablets, and desktop)
- Dynamic **theme toggle** (Light/Dark mode)
- Interactive **contact form** with validation
- Clean **Bootstrap-based design**
- Links to GitHub repositories for each project

---

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Backend (Project References):** PHP, MySQL
- **Version Control:** Git & GitHub
